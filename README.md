# 🔮 Visionary Tarot (灵视塔罗)

> 企业级架构的跨次元沉浸式互动塔罗牌应用

灵视塔罗是一个将**传统隐秘学术（塔罗牌阵、流年占星）与最前沿的机器视觉及生成式人工智能相结合**的全栈前端项目。具备极具年轻化与审美张力的 **Glassmorphism (玻璃态)** 工业级质感设计风格。

核心玩法通过实时计算机视觉 (MediaPipe) 捕捉用户的真实手势，以非物理接触的方式完成空间中的“洗牌”和“抽卡”等神圣共鸣仪式。并集成大语言模型 (Gemini AI) 以及内建快速词库，提供兼具学术严谨与神谕隐喻的深度解读。

## 🪐 特性 (Features)

*   **视觉魔法 (MediaPipe CV)**: 直接捕捉物理世界中食指与大拇指的三维特征坐标。
    *   **隔空施法**：在镜头前划出圈形运动触发宇宙洗牌池重组。
    *   **凌空提取**：用食指与拇指进行捏合手势提取出牌灵卡阵。
    *   **原力飞书**：支持在悬空状态下通过捏合并上下拖拽来操控查阅最后冗长的流式解析报告（如同使用隐形卷轴滑动）。
*   **硬件加速的 2.5D 仪式场引擎**: 使用 `framer-motion` 与 CSS3D `preserve-3d` 构建的虚拟法阵，并完全采用硬件 `MotionValue` 取代了落后的 React Re-render 以保障物理光球极高帧率滑行。支持绝绝子的实体宇宙深渊星芒卡背。
*   **双轨路由占卜引擎**:
    *   **LLM 推理流 (Gemini)**：提供基于星象学、符号学和深层心理原型的智能解析，由底层基于极速的 `AsyncGenerator` 提供毫秒级的 `Server-Sent Events (SSE)` 打字机缓冲体验。
    *   **灵界屏蔽降级系统**：当用户未能配置有效 API Key，系统无缝且极速地回退至本地塔罗知识预置短语，确保流程的自洽运转永不阻断。
*   **资源收回安全隔离**: 当关闭窗口或中断路由时执行严格全周期相机权限与缓存截获流销毁，保护用户隐私。
*   **画布离屏安全印章导出**: 内部构建了隐蔽的 2D DOM 高清树对 `html2canvas` 导出兼容，无痕绕开纯 CSS3D 导出截图花屏的技术限制，为算力祈祷结果自动印上火漆戳。

## 🛠 技术栈 (Tech Stack)

*   **视图层**: React 18
*   **开发语言**: TypeScript
*   **状态机制**: Zustand 
*   **物理架构**: Framer Motion
*   **计算机视觉**: `@mediapipe/hands`, `@mediapipe/camera_utils`
*   **工程构建**: Vite

## 🚀 启动指南 (Getting Started)

1. **环境准备**
   确保你已安装 `Node.js` (v16+ 推荐)。

2. **安装依赖**
   ```bash
   npm install
   ```

3. **进入沉浸法阵 (开发模式)**
   ```bash
   npm run dev
   ```
   > 第一项仪式需允许浏览器设备启用本地硬件摄像头权限。

## 📁 目录概览

```text
src/
├── components/   # 重用组件呈现 (光球定位、星轨背景 TarotCard 实心层)

├── hooks/        # React Hook 抽象 (凌空手势解析 useGestureEngine)
├── models/       # 接口定义契约
├── services/     # 双流 AI 微秒级释理解码器服务
├── store/        # Zustand 分发中心
├── views/        # 主层级视窗 (法阵台、启示星盘)
```
